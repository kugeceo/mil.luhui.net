南京大屠杀 Nanjing Massacre



鲁虺/撰稿/整理

南京大屠杀是抗日战争时期侵华日军屠杀中国人民的暴行之一，是震惊世界的特大惨案。以下是关于它的详细介绍：
- **事件背景**：1937年11月，淞沪会战结束后，日军分三路向南京进逼，企图占领南京，迫使中国政府投降。12月7日，日军华中方面军司令官松井石根亲自起草《攻占南京城要领》，要求部队入城扫荡。随后，日本朝香宫鸠彦亲王签署密令，要求“杀掉全部俘虏”。
- **事件经过**：1937年12月13日，日军攻占南京后，在华中方面军司令松井石根和第六师团长谷寿夫等的指挥下，以搜索抵抗军人为名，对手无寸铁的南京市民和被俘的中国军人进行了长达6周的血腥大屠杀。
- **日军暴行**
    - **大规模屠杀**：日军使用集体枪杀、活埋、刀劈、火烧等惨绝人寰的手段杀害中国军民。据战后中国审判战犯军事法庭在《谷寿夫战犯案判决书》中认定，被集体屠杀的有19万多人，零散被杀经慈善机关收埋的尸体有15万多具，总计死亡人数达30多万。
    - **奸淫暴行**：日军不分时间、地点肆意奸污妇女，上至80岁的老人，下至八九岁的幼女，有的被轮奸数十次，有的被先奸后杀。经远东国际军事法庭认定，第一个月就有2万起强奸事件。
    - **抢劫与破坏**：日军进行了大规模抢劫，抢劫物品从生产资料到生活用品无所不包。同时，工厂、商店、民宅、机关等在洗劫后还被放火烧毁，全市被焚房屋达1/3，故宫南迁文物3000箱、中央图书馆80多万册藏书等也均遭抢掠。
- **历史见证**：南京大屠杀的暴行被《纽约时报》《中央日报》《新华日报》等中外媒体广泛揭露。此外，还有大量的幸存者证言、历史照片、影像资料等，如南京大屠杀幸存者常志强、余昌祥等都亲身经历了那场浩劫，他们的回忆是南京大屠杀的重要见证。
- **纪念与反思**：1985年8月15日，侵华日军南京大屠杀遇难同胞纪念馆在江东门遇难同胞遗骨丛葬地“万人坑”遗址上落成。2014年，中国将12月13日定为南京大屠杀死难者国家公祭日。2015年，《南京大屠杀史档案》被列入《世界记忆名录》。

南京大屠杀是侵华日军公然违反国际条约和人类基本道德准则的罪恶行径，是日军在侵华战争期间无数暴行中最突出、最有代表性的一例。我们应铭记历史，以史为鉴，珍惜和平。






### 南京大屠杀概述

南京大屠杀（Nanjing Massacre），又称“南京暴行”或“南京事件”，是1937年12月13日日本军队攻占中国南京后，对中国军民实施的一场系统性暴行，持续约6周。这被视为第二次世界大战中最严重的战争罪行之一，造成约30万中国军民死亡，并伴随大规模强奸、抢掠和焚烧。事件由日本帝国陆军主导，旨在摧毁中国抵抗意志。联合国和国际社会多数承认其为反人类罪，但日本右翼势力部分否认或淡化其规模。

#### 背景
1937年7月7日卢沟桥事变爆发，中日全面战争升级。日本关东军和华中派遣军迅速推进，11月上海战役后，日军直指南京——当时中华民国首都。中国国民政府下令撤退，留下数十万军民。日军统帅松井石根下达“攻占南京”命令，但未明确禁止暴行。日军士兵受军国主义宣传影响，视中国人为“劣等民族”，导致纪律崩溃。

#### 时间线
以下是关键事件的时间线：
- **1937年12月5日**：日军第10军、第13军和第16军逼近南京外围，轰炸市区。
- **12月13日**：日军谷寿夫第6师团从中华门突入，血洗中山北路和中央路难民区，拉开屠杀序幕。
- **12月14-18日**：其他师团相继进入，集体屠杀展开，包括机枪扫射、活埋和焚烧。
- **12月下旬**：强奸高峰期，约2万妇女受害；安全区内西方人士（如约翰·马吉）记录暴行。
- **1938年1-2月**：暴行渐止，日军转向伪政权建立，但零星屠杀持续至2月底。

#### 规模与影响
- **死亡人数**：中国官方认定30万以上，包括28起集体屠杀（19万余人）和858起零散屠杀（15万余人）。受害者多为平民和放下武器的士兵。
- **暴行细节**：日军使用刺刀、机枪和火烧；强奸案达2万起，许多妇女被集体凌辱后杀害；财产被掠夺，城市三分之一焚毁。美国牧师约翰·马吉用摄影机记录轰炸和搜捕场景。
- **社会冲击**：幸存者创伤深重，南京人口锐减；事件加速中国抗日统一战线形成。

#### 国际反应
西方记者和传教士（如明妮·魏特琳）在南京安全区保护20万难民，并向国际报告暴行。1946-1948年东京国际军事法庭审判谷寿夫等战犯，认定其违反海牙公约，判处死刑。战后，中国南京审判战犯军事法庭确认屠杀事实。近年来，纪录片如《南京！南京！》和幸存者访谈进一步传播真相。

#### 后果与纪念
事件影响中日关系至今，日本教科书争议加剧紧张。1985年，中国建立侵华日军南京大屠杀遇难同胞纪念馆；2014年，联合国教科文组织列入《世界记忆名录》。每年12月13日为国家公祭日，2025年正值88周年，国际社会强调“勿忘历史”。截至2025年，南京普查记录2750多份幸存者证言，出版多部口述史。

### 东京审判细节

#### 概述
东京审判，全称为“远东国际军事法庭审判”（International Military Tribunal for the Far East），是第二次世界大战后盟国对日本战犯的审判，旨在追究日本军国主义领导人的战争责任。该审判从1946年5月3日持续至1948年11月12日，历时两年半，审判28名甲级战犯，揭露了日本从1931年侵略中国东北至1945年投降期间的侵略罪行，包括南京大屠杀等暴行。审判确立了“侵略战争罪”和“反人类罪”的先例，但也因未审判天皇裕仁和部分政治争议而被批评为“不彻底”。

#### 背景
审判源于《开罗宣言》（1943年）、《波茨坦宣言》（1945年）和日本投降书（1945年9月2日），盟国承诺对战犯“最严厉制裁”。1945年8月15日日本宣布无条件投降后，盟军最高司令官道格拉斯·麦克阿瑟于9月11日下令拘捕39名嫌疑人，最终选定28人。审判由11国法官组成（美国、苏联、中国、英国、加拿大、澳大利亚、法国、荷兰、新西兰、印度、菲律宾），美国主导，但中国法官梅汝璈争取到中国座次优先（按受降签字顺序）。

#### 时间线
以下是关键事件的时间线：

| 时间          | 事件                                                                 |
|---------------|----------------------------------------------------------------------|
| 1945年8月15日 | 日本宣布无条件投降。 |
| 1945年9月11日 | 麦克阿瑟下令拘捕战犯嫌疑人。 |
| 1946年1月19日 | 公布《远东国际军事法庭宪章》。 |
| 1946年4月29日 | 对28名甲级战犯提起公诉。 |
| 1946年5月3日  | 审判正式开庭。 |
| 1948年11月4-12日 | 宣读判词。 |
| 1948年12月23日 | 7名死刑犯执行绞刑。 |
| 1952年4月28日 | 盟国占领结束，日本主权恢复。 |
| 1956年3月前   | 所有受刑者获释或假释。 |

#### 被告名单
审判28名被告，主要为日本军政要员（2人审判中死亡：松冈洋右、永野修身；1人精神疾病免诉：大川周明）。以下是主要被告分类：

| 类别 | 主要被告（部分） |
|------|------------------|
| **文官/外交官** | 东乡茂德、重光葵、广田弘毅、木户幸一、平沼骐一郎 |
| **陆军将领** | 东条英机、土肥原贤二、板垣征四郎、松井石根、武藤章 |
| **海军将领** | 嶋田繁太郎、铃木贞一、梅津美治郎 |
| **其他** | 荒木贞夫、小矶国昭、南次郎 |

#### 指控罪名
根据法庭宪章，指控分为三类：甲级（侵略战争罪）、乙级（战争罪）、丙级（反人类罪）。公诉书列出55条罪状，最终审理10条主要指控，包括：
- 阴谋发动侵略战争（共同计划）。
- 对中国（1931年九一八事变起）、美国（珍珠港事件）、英国、荷兰、法国、苏联的侵略战争。
- 未防止或下令战俘不人道待遇（如南京大屠杀、巴丹死亡行军）。

#### 审判过程
审判在东京市谷区原陆军士官学校旧址进行，持续818天，开庭818次，证人出庭419人，书面证人779人，证据4336份，英文记录达4.8万页。过程包括：
- **起诉阶段**：首席检察官约瑟夫·B·基南宣读42页起诉书，历数日本罪行。中国检察官向井久（后由高文彬等接替）负责中国部分，检控南京大屠杀等。
- **辩护阶段**：日本律师团辩称“自卫战争”，但证据确凿。
- **法官审议**：首席法官澳大利亚人威廉·韦伯主持，梅汝璈坚持中国优先。印度法官帕尔等3人反对，认为违反“不溯及既往”原则。
- 审判揭露暴行，如南京大屠杀（30万人死亡）、慰安妇制度。

#### 判决结果
1948年11月12日宣判，25人定罪（3人无罪：贺屋兴宣、松冈洋右、永野修身已故）。结果如下：
- **死刑（绞刑，12月23日执行）**：东条英机、广田弘毅、土肥原贤二、板垣征四郎、木村兵太郎、松井石根、武藤章、桥本欣五郎、平沼骐一郎、荒木贞夫、小矶国昭、南次郎（7人执行，桥本等因健康暂缓）。
- **无期徒刑**：重光葵、木户幸一、冈敬纯、大川周明（后减刑）。
- **有期徒刑**：梅津美治郎（20年）、铃木贞一（20年）、东乡茂德（20年）、佐藤贤了（15年）、星野直树（终身）。
- **无罪**：贺屋兴宣。

#### 历史影响
东京审判与纽伦堡审判并列确立国际刑法先例，推动联合国《防止及惩治灭绝种族罪公约》（1948年）。对中国而言，肯定了抗日战争正义，但未彻底追究（如天皇免诉）。日本右翼视其为“东京审判史观”，争议持续；2025年抗战胜利80周年，纪念活动强调历史真相。审判档案（如NARA收藏）达数十万页，支持后续研究。



### 梅汝璈在东京审判中的作用

#### 概述
梅汝璈（Mei Ru'ao，1904-1973）是中国著名法学家、外交家，二战后代表中国出任远东国际军事法庭（即东京审判）法官。他是法庭11位法官中唯一的中国代表，任中国法律代表团团长和首席法官。在1946-1948年的审判中，梅汝璈发挥了关键作用，不仅维护了中国的民族尊严，还推动了对日本侵华罪行的追究和量刑公正。他的努力确保了中国在国际舞台上的发言权，尤其在南京大屠杀等暴行证据的呈现上。

#### 主要作用与贡献
梅汝璈在审判中的作用体现在多个关键环节，以下是主要方面：

1. **争取中国法官席位优先**：
   - 审判伊始，梅汝璈代表中国争取到法官席位安排的优先权。根据受降签字顺序，中国法官位居首席，这象征着中国作为战胜国的尊严。他在席位之争中强调中国受害最深，成功说服盟国接受这一安排。

2. **推动侵华罪行证据收集与呈现**：
   - 作为最早提出系统研究南京大屠杀的法官之一，梅汝璈呼吁中国学术界填补相关研究空白。他在审判中收集并呈现大量日军暴行证据，包括南京大屠杀中的砍头、挖心、火烧等残酷细节，这些证据直接用于对谷寿夫、松井石根等战犯的指控。

3. **起草判决书侵华部分**：
   - 在起草判决书时，梅汝璈主张中国人受害最深、最具发言权，法庭接受其提议，由他负责撰写有关日军侵华罪行的章节。这部分判决书详细记录了从九一八事变到南京大屠杀的侵略事实，成为审判的核心证据。

4. **坚持死刑量刑与伸张正义**：
   - 梅汝璈在审议中强烈主张对东条英机、土肥原贤二、松井石根等主要战犯判处死刑。他根据审判中收集的两年暴行证据，反驳辩护方的“自卫战争”论点，确保7名甲级战犯被执行绞刑。他的日记记录了审判现场细节，如东条英机等“侵华老手”庭审场景，体现了其对正义的追求。

5. **维护家国情怀与国际影响**：
   - 尽管享受丰厚待遇，梅汝璈心系祖国，常从报纸关注国内动态。他一生追求正义，推动东京审判确立“侵略战争罪”先例，对后世国际刑法有深远影响。回国后，周恩来总理亲自接见，赞其为中国人民争得尊严。

#### 历史评价
梅汝璈的角色被视为中国在东京审判中的“脊梁”，他克服语言障碍和盟国内部分歧，体现了中国法学家的风骨。2025年抗战胜利80周年之际，其故居和日记手稿公开，进一步凸显其贡献。尽管审判后他遭受政治迫害，但其正义追求永载史册。

### 梅汝璈日记关键摘录

梅汝璈的日记手稿是其在东京审判期间（1946年3月20日至5月13日）的珍贵记录，仅存200多页、约5万字，记录了审判开庭前的准备、内心情感、对战犯的义愤以及对祖国的忧思。这些摘录反映了他作为中国法官的使命感、民族自强意识和对正义的追求。以下按日期或主题整理关键摘录，基于可靠历史档案和后人整理。

#### 1946年4月7日：人与人互助的动人一幕
“在我们车子挣扎(陷入泥中)的时候，那些气力较大的日本人都很踊跃地一齐来帮我们推拉，大卖气力。他们明知我们是中国人，但丝毫没有愠色。不一会儿，一部巨型载重车来了，载着三四十个美国士兵。他们看见情景不佳，一跃而下，大家齐心合力，把那辆日本卡车(也陷入泥中)也推动了。当时三辆车上的人都很感愉快，大家心心相印，一团和气。这是很动人的一幕，在我脑海里留有深刻的印象。”

#### 1946年4月16日：对国家国际地位的忧虑
“我们的国际地位由于八年的浴血苦战，已经赚得很多，倘使自己不争气，眼见这个地位就要堕落了。”

#### 1946年4月19日：与美国法官的对话
“他(美国法官希金士)说一般美国人最感亲切、爱好的民族是中国人。他说的并非虚伪，我想大体实际是如此。不过我们得争气才对，否则‘仇者所快’必然是‘亲者所痛’。”

#### 1946年5月3日：审判开庭首日，东条英机等战犯入庭
“今天是远东国际军事法庭正式开演的第一天，也是我参加客串这出富于历史性戏剧的开锣第一幕。虽然每个法官席上都摆了一张很清楚的犯人照片，而且这照片是依照他们的座次排列的。我只注意到坐在中央的东条和肥圆圆脸的土肥原……都是板着面孔，佯作镇静，尤其是东条，简直一动不动，像石膏塑的人一般。”

#### 1946年5月3日：对战犯的深切愤恨
“这些人都是侵华老手，毒害了中国几十年，我数百万数千万同胞曾死于他们的手中，所以我的愤恨便是同胞的愤恨。我今天能高居审判台上来惩罚这些元凶巨憝，都是我千百万同胞的血肉换来的，我应该警惕！我应该郑重！”

#### 无具体日期：对战犯罪行的愤怒与法治坚持
“法治时代，必须先审后斩，否则我真要先斩他几个，方可雪我心头之恨！”

#### 无具体日期：反复出现的民族自强箴言
“中国人还得争气才行！”（此句在日记中多次出现，反映梅汝璈对祖国未来的忧思和警醒。）

#### 审判准备期：座位之争（日记记录）
“开庭前预演时就发生了争议，因为庭长韦伯宣布入场的顺序为美、英、中、苏、加、法、新、荷、印、菲，把中国法官的座次排在英国之后。梅汝璈立即对这一安排提出强烈抗议：‘如论个人之座位，我本不在意。但既然我们代表各自国家，我认为法庭座次应该按日本投降时各受降国的签字顺序排列才最合理。首先，今日系审判日本战犯，中国受日本侵害最烈，且抗战时间最久、付出牺牲最大，因此，有八年浴血抗战历史的中国理应排在第二，再者，没有日本的无条件投降，便没有今日的审判，按各受降国的签字顺序排座，实属顺理成章。’接着，他愤然脱下象征着权力的黑色丝质法袍，拒绝‘彩排’，他说：‘今日预演已有许多记者和电影摄影师在场，一旦明日见报便是既成事实。既然我的建议在同仁中并无很大异议，我请求立即对我的建议进行表决。否则，我只有不参加预演，回国向政府辞职。’由于他的据理力争，庭长当即召集法官们表决，结果入场顺序和法官座次按日本投降各受降国签字顺序安排。”
